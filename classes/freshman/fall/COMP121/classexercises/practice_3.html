Memorandum

Below are the various courses to be offered during the 2013-14 academic year. Below each course is the course learning objectives and outcomes. All course information can be found on the Blackboard Learning Management System. http://bblearn.linfield.edu

Courses for 2013-14 Academic Year

COMP121Intro to Internet and WWWFall 2013

COMP121 Objectives and Learning Outcomes

  • Understand basic UNIX and Windows operations on files and folders.
  • Manipulate and optimize an image for web display
  • Follow the recommended web design guidelines when designing a web page
  • Define basic HTML tags
  • Create style sheets that configure various parts of a web page
  • Utilize the "cascade" in CSS and validate CSS
  • Use a web page design editor to build web pages
COMP260Database Management SystemFall 2013

COMP260 Objectives and Learning Outcomes

  • Have an understanding of the principles of database systems, industry standard SQL, and database analysis, design, and implementation
  • Have the experience of designing and developing database applications, using industry standard software.
  • Experience the difficulties of working in teams and investigated techniques for overcoming those difficulties.
  • Create precise and informative documents for each stage of database development.
  • Be exposed to industry standard software tools used in database development.
  • Learn to incorporate design-for-maintanability considerations through approrpiate project work.
  • Be exposed to the design and implementation of database technology that is widely-used in industrial and commercial environments.
  • Acquire a practical experience that will prepare students to lead or participate in large database projects.
COMP305Software EngineeringSpring 2014

COMP305 Objectives and Learning Outcomes

  • Describe a phased approach to information systems development and describe cross-life-cycle activities that overlap the entire cycle.
  • Present a technical material to a group.
  • Create precise and informative documents for each stage of software development.
  • Apply several approaches to software design such as data flow oriented, data structure oriented, and object oriented techniques.
  • Understand the importance of quality assurance and reliability of software systems.
  • Understand the process of graphic user interface (GUI) and dialogues design using Visual C#
  • Access relational data using ADO.NET
  • Apply object-oriented techniques to Visual C# or VB.NET programs
COMP485Special Topics in Computer ScienceSpring 2014

COMP485 Objectives and Learning Outcomes

  • Install, configure, and maintain Web server and an Intranet or Internet Web site
  • Create and publish Web pages using a web page editor
  • Create dynamic web pages to capture, retrieve, and display information via a database using PHP and ASP.NET
  • Develop applications for mobile devices
